    First Project Tutorial¶

    This comprehensive tutorial walks you through creating your first Proxmox infrastructure project with Proxmox-MPC. You'll learn the complete workflow from initialization to deployment.

    üéØ What You'll Learn¶

    By the end of this tutorial, you'll have:

    • ‚úÖ Created and configured a new Proxmox project workspace
    • ‚úÖ Connected to your Proxmox server with proper authentication
    • ‚úÖ Synchronized existing infrastructure into IaC files
    • ‚úÖ Created new VMs using declarative configuration
    • ‚úÖ Generated and executed infrastructure tests
    • ‚úÖ Deployed changes to your Proxmox server

    Time Required: 30-45 minutes
    Prerequisites: Installation and Authentication completed

cd ~/my-proxmox-datacenter
